## 2️⃣ INFORMATION (정보 층위)

### 가공된 정보 및 통계

#### A. 혈연 불평등 (가문 중심)
```
✅ 파레토 분석
  - 상위 10% 가문(77개) → 34.87% 급제
  - 상위 20% 가문(155개) → 42.50% 급제

✅ 지니계수
  - 0.7915 (극심한 불평등)
  - 현대 소득 불평등의 2.3배

✅ 세과(世科) 가문
  - 99개 가문이 3대 이상 연속 급제
  - 평균 450년 지속
```

#### B. 지연 불평등 (지역 중심)
```
✅ Representation Index (RI)
  지역          RI      해석
  경기/한양     4.91    인구 대비 4.91배 과대표
  평안          1.13    약간 과대표
  전라          0.39    심각한 과소대표
  제주          0.08    거의 배제

✅ 등위(성적) 분석
  - ANOVA: p < 0.001 (지역 차이 존재)
  - 그러나 Tukey HSD: 대부분 유의하지 않음
  - 회귀분석: 지역 더미 변수 p > 0.05
  → 능력 차이 아님, 접근성 차이임

✅ 시대별 변화
  - 1390년대: RI = 10 (극단적 수도 독점)
  - 1500-1600년: RI ≈ 6 (사림 성장기)
  - 1700년 이후: RI < 4 (지방 확대)
```

## 3️⃣ KNOWLEDGE (지식 층위)

### 패턴 발견 및 인과관계

#### 핵심 발견 1: 이중 차별 구조
```
┌─────────────────────────────────────┐
│  조선시대 과거제 = 이중 필터 시스템    │
└─────────────────────────────────────┘

제1 필터: 지역 (접근성)
  ↓
  경기/한양 출신만 시험장에 대거 도달
  (인프라, 비용, 정보, 추천권)
  
제2 필터: 혈연 (문화자본)
  ↓
  수도 내에서도 소수 명문가가 독점
  (지니계수 0.79)
```

#### 핵심 발견 2: 능력 vs 구조
```
✅ 능력(성적)은 지역 차이 없음
  → ANOVA는 유의하나, 실질적 차이는 미미
  → 지방 출신도 시험에 도달하면 성적은 비슷

❌ 그러나 합격자 수는 압도적 차이
  → 경기/한양 4.91배 과대표
  → 전라/강원/함경 0.39배 과소대표
  
📌 결론: 능력의 문제가 아니라 접근성의 문제
```

#### 핵심 발견 3: 시대별 진화
```
[1기] 1390-1450년: 극단적 중앙 독점
  - RI = 10 (수도가 10배 과대표)
  - 건국 귀족 중심
  - 혈연: 지니 0.49 (상대적 평등)

[2기] 1450-1650년: 사림 성장기  
  - RI = 6 (완화 추세)
  - 지방 사림 진출 증가
  - 혈연: 지니 0.61 (불평등 증가)

[3기] 1650-1850년: 문벌 사회 고착
  - RI = 4 (지역 완화 지속)
  - 그러나 혈연: 지니 0.72 (극심화)
  - 지역은 열렸으나, 가문은 닫힘
```

#### 핵심 발견 4: 혈연과 지연의 결합 효과
```
┌──────────────────────────────────────────┐
│ 최대 수혜자 = 경기/한양 + 명문 세과 가문  │
└──────────────────────────────────────────┘

예시:
  - 이 전주(全州) 870명 중 329명이 한성 거주 (37.8%)
  - 권 안동(安東) 368명 중 지역 안동 거주는 26명 (7%)
    → 대부분 한양 이주 후 급제

📊 이중 특권의 누적 효과
  경기 출신 × 명문가 = 4.91 × (1/0.79) ≈ 6배 특혜
```

#### 핵심 발견 5: 구조적 장벽의 정체
```
[지연 장벽] 지방 출신의 어려움
  1. 교육 인프라 부족 (성균관, 사학은 한양)
  2. 시험 응시 비용 (거주비, 이동비)
  3. 정보 비대칭 (출제 경향, 추천 네트워크)
  4. 정치적 차별 (평안, 함경 등)

[혈연 장벽] 신진 가문의 어려움
  1. 문화자본 부족 (가학, 장서, 학문 전통)
  2. 경제자본 부족 (토지, 노비, 준비 비용)
  3. 사회자본 부족 (혼인망, 스승 네트워크)
  4. 음서 배제 (고위 관직 조상 없음)

💡 두 장벽의 교차점
  → 지방 + 비명문가 = 거의 불가능
  → 수도 + 명문가 = 세습적 독점
```

## 4️⃣ WISDOM (지혜 층위)

### 역사적 교훈 및 현대적 함의

#### 지혜 1: "공정성"의 이중성

```
조선시대 과거제가 던지는 질문:

"시험 그 자체는 공정했는가?"
  → YES. 성적은 지역/가문과 무관

"시험에 대한 접근은 공정했는가?"  
  → NO. 지역(RI 4.91배) + 혈연(지니 0.79)

┌──────────────────────────────────────┐
│  형식적 공정 ≠ 실질적 공정            │
│  Formal Fairness ≠ Substantive Equity │
└──────────────────────────────────────┘
```

#### 지혜 2: 현대 사회와의 패러럴

| 조선시대 과거제 | 현대 한국 교육/입시 |
|----------------|---------------------|
| **경기/한양 4.91배 과대표** | **서울 출신 SKY 대학 진학률 3-4배** |
| 성균관, 사학 집중 | 특목고, 자사고 집중 (강남, 서초) |
| 시험 응시 비용 | 입시 사교육 비용 (연 평균 400만원) |
| 추천 네트워크 | 입학사정관제, 학생부종합전형 |
| **지니계수 0.79 (혈연)** | **전문직 세습 (의사 자녀의 의대 진학률 20배)** |
| 세과(世科) 가문 | "금수저" 엘리트 재생산 |
| 음서(蔭敍) | 특례입학, 특기자 전형 |

```
📌 500년이 지났지만, 구조는 반복된다
   → 능력주의(Meritocracy)의 허상
   → 기회의 불평등이 결과의 불평등을 만든다
```

#### 지혜 3: 정책적 시사점

```
[역사의 교훈]
  지역 불평등(지연)은 시간이 지나며 완화됨
  - RI: 10 → 4 (60% 감소)
  - 지방 교육 인프라 확대 효과

  그러나 가문 불평등(혈연)은 악화됨  
  - 지니: 0.49 → 0.72 (47% 증가)
  - 문화자본 세습 강화

📍 정책 제언
  1. 지역 균형 발전 (인프라 투자)
     → 조선도 지방 향교/서원 확대로 완화
     
  2. 문화자본의 공공화
     → 교육은 "가문"이 아니라 "국가"가
     → 평등한 출발선 보장
     
  3. 평가의 다원화
     → 단일 시험(과거/수능)의 한계
     → 다양한 경로 개방
```

#### 지혜 4: 디자인 윤리

```
💡 디자이너를 위한 질문

"우리가 만드는 시스템은 누구를 위한 것인가?"

[Bad Design]
  - 형식적 공정성만 강조
  - "시험은 똑같이 치렀다"
  - 결과의 불평등을 개인 책임으로 전가

[Good Design]  
  - 접근성을 설계에 포함
  - "시험장에 도달할 수 있는가?"
  - 구조적 장벽을 제거하는 설계

┌─────────────────────────────────────┐
│ 공정한 시스템 ≠ 공정한 결과          │
│                                     │
│ 공정한 결과 = 공정한 시스템          │
│             + 공정한 접근성          │
│             + 구조적 지원            │
└─────────────────────────────────────┘
```

### 제안 4: 인터랙티브 설치 미술

```
🎪 "급제의 문" - 체험형 설치

[컨셉]
  관람자가 직접 "과거 시험 응시"를 체험
  본인의 선택에 따라 결과가 달라지는 인터랙티브

[구조]
  - 입구: 터치 스크린
    "당신의 출신지는?" → 8개 선택지
    "당신의 가문은?" → 명문/중간/신진 선택
    
  - 복도: 8개의 문
    각 문마다 "급제 확률" 표시
    예) 경기 + 명문 = 6.2%
        전라 + 신진 = 0.5%
        
  - 선택한 문을 열면:
    [성공] LED 초록불, 축하 사운드, 합격증 프린트
    [실패] LED 빨간불, 낙방 문구, 재시험 안내
    
  - 확률은 실제 데이터 기반
    (RI × 가문 급제율)
    
  - 출구: 대형 스크린
    "오늘의 급제 현황"
    - 실시간 방문자 통계
    - 출신지별 합격률 그래프
    - "가장 불리한 조합은?"
```
