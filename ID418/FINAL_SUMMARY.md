## 🔑 핵심 발견 사항 요약

### 1️⃣ 가문별 집중도 분석 (파레토 & 로렌츠)

#### 📈 파레토 법칙 검증
```
상위 10% 가문 (77개) → 급제자의 34.87% 배출
상위 20% 가문 (155개) → 급제자의 42.50% 배출
```

#### 📊 로렌츠 곡선 & 지니계수
```
지니계수: 0.7915 (극심한 불평등 🔴)

비교:
• 현대 한국 소득 지니계수: ~0.35
• 조선시대 과거제: 0.7915
• 차이: 2.3배 더 불평등!
```

**💡 해석:**
- 과거제는 형식적으로 능력주의였으나, 실제로는 **소수 명문가의 독점 구조**
- 현대 소득 불평등보다 **2.3배 심각한 수준**
- "만인에게 열린 기회"는 **이상**일뿐, **실제**는 폐쇄적

### 2️⃣ 세대 연속성 분석 (세과 가문)

#### 👨‍👩‍👧‍👦 세대 연속성 분류
```
• 비연속 (단발/간헐): 451개 (57.9%)
• 2대 연속 추정: 229개 (29.4%)
• 3대 연속 추정: 79개 (10.1%)
• 4대+ 연속 추정: 20개 (2.6%)
```

#### 🏛️ 최상위 세과 가문 (3대 이상 연속)
```
총 99개 가문 → 전체의 12.7%

TOP 3:
1. 한 청주(淺): 292명, 497년간 (1396-1893)
2. 서 대구(大丘): 143명, 448년간 (1444-1892)
3. 송 여산(礪山): 107명, 490년간 (1402-1892)

평균 지속 기간: 약 450년
```

**💡 해석:**
- 일부 가문은 **조선 건국부터 말기까지 500년간** 급제자 배출
- 이는 단순한 개인 능력이 아닌 **가문 단위의 문화자본 세습**
- **세습적 학문 전통, 경제력, 사회적 네트워크**의 복합 작용

### 3️⃣ 시기별 혈연 영향력 변화 (시계열 분석)

#### 📅 시대별 지니계수 변화

| 시대 | 지니계수 | 상위10% 집중도 | 평균 급제자/가문 |
|------|----------|----------------|-----------------|
| **조선 초기** (1392-1449) | 0.4895 | 38.99% | 2.99명 |
| **조선 전기** (1450-1549) | 0.6058 | 48.20% | 5.47명 |
| **조선 중기** (1550-1649) | 0.6465 | 49.93% | 7.28명 |
| **조선 후기 전반** (1650-1749) | 0.6925 | 56.17% | 8.77명 |
| **조선 후기 후반** (1750-1849) | **0.7162** | **58.68%** | **10.05명** |
| **조선 말기** (1850-1910) | 0.6763 | 55.50% | 7.30명 |

#### 📈 변화율
```
지니계수:
  0.4895 (초기) → 0.7162 (후기) = +38.1% 증가

상위 10% 집중도:
  38.99% (초기) → 58.68% (후기) = +16.5%p 증가

평균 급제자/가문:
  2.99명 (초기) → 10.05명 (후기) = +236% (3.4배)
```

**💡 해석:**
- 시간이 지날수록 **불평등 심화** + **명문가 독점 강화**
- 조선 후기 = **"문벌 사회"의 고착화**
- 신진 가문의 진입 장벽 ↑ → 사회 이동성 ↓

### 4️⃣ 혼인 관계망 분석 (향후 과제)

#### 🚧 현재 데이터의 한계
```
한국학중앙연구원 데이터에 미포함:
❌ 배우자 정보 (처가)
❌ 모친 정보 (외가)
❌ 직계 조상 (부, 조부, 증조부)
❌ 음서(蔭敍) 여부
```

#### 🔮 향후 연구 방향
1. **족보 데이터 통합**
   - 급제자의 4대조 정보
   - 외가 및 처가 본관
   - 형제, 사촌 등 방계 친척

2. **네트워크 분석 (SNA)**
   - 성관 간 혼인 네트워크
   - 중심성 지표 (degree, betweenness, closeness)
   - 커뮤니티 탐지 (통혼권 형성)

3. **예상 발견**
   - 특정 명문가들 간의 밀집 혼인망
   - "통혼권(通婚圈)" → 엘리트 계층의 폐쇄성
   - 중심 가문과 주변 가문의 위계 구조

## 🎓 학술적 의의

### 1. 과거제의 이중성 입증 (정량적 증거)

| 형식 (建前) | 실제 (本音) |
|------------|-----------|
| ✅ 능력주의 | ❌ 지니계수 0.79 |
| ✅ 만인에게 열린 기회 | ❌ 상위 10%가 35% 독점 |
| ✅ 유교적 이상 | ❌ 시대별 불평등 심화 |

### 2. 문화자본의 세습 메커니즘

```
학문적 전통 (가학, 장서)
    +
경제적 안정 (토지, 노비)
    +
사회적 네트워크 (혼인, 관직)
    ↓
세습적 급제 시스템
    ↓
"문벌 사회" 형성
```

### 3. 조선 사회의 구조적 특징

**사회 이동성의 제한:**
- 신진 가문 진입 장벽 ↑
- 기존 명문가 지위 세습 ↑
- "개천에서 용 나기" 점점 어려워짐

**역사적 의미:**
- 조선 초기: 다양한 가문 진출 (지니 0.49)
- 조선 중기: 특정 가문 부상 (지니 0.65)
- 조선 후기: 명문가 독점 (지니 0.72)
- 조선 말기: 문벌 사회 고착화 (지니 0.68)

## 💡 현대적 함의

### 교육 불평등과의 유사성

| 조선시대 | 현대 |
|---------|------|
| 지니계수 0.79 | SKY 대학 지역/계층별 격차 |
| 문화자본 세습 | "금수저" vs "흙수저" |
| 명문가 독점 | 전문직 가문 (의사, 법조인) |
| 세과(世科) | 세습 엘리트 |
| 음서(蔭敍) | 특목고, 자사고 |

### 시사점
1. **공정성의 본질**
   - 형식적 공정 ≠ 실질적 공정
   - 기회의 평등 ≠ 결과의 평등

2. **문화자본의 중요성**
   - 부르디외(Bourdieu) 이론의 역사적 검증
   - 가정 환경, 교육 배경의 결정적 영향

3. **정책적 과제**
   - 교육 기회 확대 정책
   - 사회적 약자 지원 제도
   - 진정한 능력주의 실현
