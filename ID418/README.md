# 조선시대 과거제 급제자 혈연관계 분석 프로젝트

## 프로젝트 개요

이 프로젝트는 한국학중앙연구원의 조선시대 과거 급제자 데이터를 활용하여 **혈연관계가 과거 급제에 미친 영향**을 정량적으로 분석합니다.

## 데이터 소스

- 문과(文科) 급제자: 30,302명
- 무과(武科) 급제자: 39,713명
- 사마시(司馬試) 급제자: 84,782명
- 잡과(雜科) 급제자: 15,630명

출처: 한국학중앙연구원 (2020년 9월 29일)

## 분석 방법론

### 1단계: 자료 수집 및 파싱

XML 형식의 원본 데이터를 파싱하여 구조화된 DataFrame으로 변환합니다.

**주요 추출 정보:**
- 개인 정보: 이름, 자(字), 호(號), 시호(諡號)
- 시험 정보: 시험명, 시험년, 왕대, 등급, 등위
- 가계 정보: 본관(本貫), 거주지
- 생몰년: 생년, 졸년

### 2단계: 데이터베이스 구축 및 혈연 관계 정의

**혈연 관계 정의:**
- **성관(姓貫)**: 성씨 + 본관 조합 (예: 이 전주, 권 안동)
- **세과(世科)**: 동일 성관에서 2대 이상 연속 급제한 가문
- **세대 간격**: 20-40년을 한 세대로 간주

### 3단계: 분석 지표

#### 3.1 본관별 집중도 분석
- 총 779개 성관 중 상위 10%(77개)가 전체 급제자의 **34.87%** 차지
- 특정 가문의 독점적 급제 현상 확인

**상위 5개 성관:**
1. 이 전주(全州): 870명
2. 권 안동(安東): 368명
3. 윤 파평(坡平): 346명
4. 한 청주(淸州): 292명
5. 홍 남양(南陽): 292명

#### 3.2 세과(世科) 가문 분석
- 3대 이상 연속 급제한 가문: **99개**
- 이는 전체 779개 성관 중 약 12.7%

**주요 세과 가문:**
| 성관 | 총급제자수 | 연속세대 | 활동기간 |
|------|-----------|---------|---------|
| 한 청주 | 292명 | 3대 | 1396-1893 (497년) |
| 서 대구 | 143명 | 3대 | 1444-1892 (448년) |
| 송 여산 | 107명 | 3대 | 1402-1892 (490년) |
| 정 나주 | 55명 | **4대** | 1432-1891 (459년) |
| 최 경주 | 45명 | **4대** | 1393-1891 (498년) |

#### 3.3 시대별 변화 분석

**조선 전기 (1392-1494):**
- 급제자 수: 1,955명
- 성관 수: 411개
- 평균 급제자/성관: **4.76명**
- 특징: 상대적으로 다양한 가문에서 급제

**조선 중기 (1494-1608):**
- 급제자 수: 2,842명
- 성관 수: 449개
- 평균 급제자/성관: **6.33명**
- 특징: 가문 집중도 증가 시작

**조선 후기 (1608-1910):**
- 급제자 수: 10,354명
- 성관 수: 557개
- 평균 급제자/성관: **18.59명**
- 특징: 특정 가문의 급제 독점 심화 (**3배 증가**)

이는 조선 후기로 갈수록 **혈연과 가문 배경이 과거 급제에 더 큰 영향**을 미쳤음을 시사합니다.

#### 3.4 지역별 분포
- 한성(서울) 출신: 3,714명 (약 12.3%)
- 정주: 202명
- 안동: 156명

중앙(한성)과 특정 지방 거점(안동, 상주, 충주 등)의 높은 비율은 교육 인프라 및 유력 가문의 지역적 집중을 보여줍니다.

## 분석 결과의 의미

### 1. 과거제의 이중성

조선 과거제는 **능력주의(meritocracy)**를 표방했으나, 실제로는:
- 특정 가문의 세습적 급제 패턴 존재
- 조선 후기로 갈수록 가문 독점 심화
- 평균 급제자/성관 비율이 4.76명 → 18.59명으로 증가

### 2. 세과(世科) 가문의 영향력

- 99개 세과 가문이 장기간(평균 400년 이상) 급제자 배출
- 이는 단순한 개인의 능력을 넘어 **가문의 문화적·경제적 자본**이 작용했음을 의미
- 족보, 가승, 학문적 전통, 경제력 등의 복합적 요인

### 3. 사회적 이동성의 제한

- 상위 10% 성관이 전체의 35% 차지
- 신진 가문의 진입 장벽이 시대가 지날수록 높아짐
- 조선 후기 "문벌(門閥)" 사회의 형성

## 연구의 한계 및 향후 과제

### 현재 분석의 한계:
1. **직계 혈통 정보 부족**: 현재 데이터는 성관(본관) 기준이며, 부, 조부, 증조부 등의 직계 정보는 미포함
2. **음서 정보 부재**: 과거가 아닌 음서로 등용된 경우의 데이터 부족
3. **혼인 관계 미반영**: 외가, 처가 등 통혼망(通婚網)을 통한 네트워크 효과 미분석

### 향후 개선 방안:
1. **족보 데이터 통합**: 개별 족보를 분석하여 4대조 정보 구축
2. **네트워크 분석**: 성관 간 혼인 관계를 네트워크로 시각화
3. **관직 데이터 연계**: 급제 후 관직 이력을 분석하여 가문의 실질적 권력 측정
4. **머신러닝 적용**: 급제 예측 모델을 통해 가문 배경의 영향력 정량화

## 학술적 의의

이 분석은 조선시대 과거제가:
1. **제도적 이상**과 **실제 운용** 간의 괴리가 존재했음을 수치로 입증
2. **사회 계층 이동**의 통로였지만 동시에 **기득권 재생산** 메커니즘으로도 작동
3. **유교적 학문 전통**이 가문 단위로 전승되며 문화자본으로 축적되었음을 시사
